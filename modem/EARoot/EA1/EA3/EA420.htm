A ModelElement that is encoded using the Scaleable Vector Graphics (1.1) standard. <br/><br/>An SvgElement can be any graphical element that is allowed by SVG v1.1 - e.g.<br/><br/>The ‘rect’ element<br/>The ‘circle’ element<br/>The ‘ellipse’ element<br/>The ‘line’ element<br/>The ‘polyline’ element<br/>The ‘polygon’ element<br/><br/>If the element is a group of other elements, then the SvgElementGroup should be used.<br/>
