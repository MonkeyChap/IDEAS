Appears In: Singletons, Doubletons, etc.
<p>Singletons are types which have only one member. Doubletons are types which have two members. Tripletons are types which have three members. Quadrupletons are types which have four members. Quintupletons are (not surprisingly) types which have five members.<br/></p><p><br/></p><p>These Types are intended for re-use across the model. There may be a need for Types which have a greater fixed number of members, and these should be created following this pattern, as required.<br/></p>
