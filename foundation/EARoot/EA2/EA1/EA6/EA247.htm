<p>A powertype is the set of all subsets of a given Type. It is quite easy to confuse a Powertype with Type whose members are Types (a Type of Types). A Powertype is a special Type of Types which consists of all the possible subsets of a given Type.<br/></p><p><br/></p><p>For example, if we have a Type "Pump", its powertype would be "PumpType". This powertype would consist of all possible types of pump - e.g. "centrifugal pump", "reciprocating pump", "peristaltic pump", "Acme Pumps Model No 1234".<br/></p><p><br/></p><p>Because the Powertype is the set of all subsets, those can include such Types as "centrifugal pumps and peristaltic pumps", "all pumps that aren't peristaltic", "all blue coloured pumps", etc.<br/></p><p><br/></p><p>For more information on this subject, see Cantor's Theorem - http://en.wikipedia.org/wiki/Cantor's_theorem<br/></p>
